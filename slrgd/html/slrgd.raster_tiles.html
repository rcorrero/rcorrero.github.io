Inherits from torch class Dataset. Handles input of training samples to torch models.

Parameters

  • data_dict (dict) – A dataset dictionary produced by the make_data function.

  • mode (str, default=train) – Determines whether to return the tuple (X, y) (when mode=train) or X (when mode=test).

  • C (int, default=4) – Number of channels for each sample.

  • size (int, default=5) – The width and height of each sample.

  • img_max (int, default=10000) – The maximum attainable value for the samples. For Planet imagery, reflectances (which range from 0 to 1) are multiplied by 10000, resulting in images with integer values ranging from 0 to 10000. These values may be too large for gradient calculations.

Extracts image tiles (or “patches”) of size (size, size) from arr. The spacing between patches is controlled by stride, which should almost always be set equal to size.

Parameters

  • arr (np.ndarray) – The target image, loaded as a numpy array.

  • size (int) – The desired extent for the image patches.

  • stride (int) – The desired spacing between adjacent patches.

slrgd.raster_tiles.make_raster_tiles.make_raster_features(filepaths: Union[list, dict], make_labels: bool, size: int, stride: int, feature_save_dir: str, label_save_dir: Optional[str] = None, flatten_patches: bool = False) → None

A helper function to create image tiles and training vectors from rasters. Saves the generated pytorch tensors to save_dir. Files must be in .tif format.

Parameters

  • filepaths (list or dict) – If make_labels = False, then a list containing the paths to the raster files to be processed. Otherwise a dict with keys corresponding to images and list values containing all label raster files associated with that image.

  • make_labels (bool) – Whether or not to make labels for the sample. If True, then filepaths should be a dict.

  • size (int) – The width and height of the tiles generated.

  • stride (int) – The spacing between tiles. If stride < size then the tiles will overlap.

  • feature_save_dir (str) – Path to the directory where features will be saved.

  • label_save_dir (str, default = None) – Path to the directory where labels will be saved.

  • flatten_patches (bool, default = False) – Whether patches, which are C imes size imes size in shape, should be flattened into C(size)(size) feature vectors.

Return type

None
