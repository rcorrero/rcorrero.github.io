slrgd.scripts.gdal_scripts.rasterize module

slrgd.scripts.gdal_scripts.rasterize.create_labels(sample_file, vector_file, attribute='FID', otype='MEM', fname_out='')

Rasterise a vector layer vector_file using a sample raster file (sample_file) to provide extent and projection. You can select an attribute in vector_file (by default, FID), and you can either do the processing in memory, or dump to disk in any GDAL supported format.

Parameters

  • sample_file (str) – A raster with a suitable geotransform, projection etc.

  • vector_file (str) – The vector file you want rasterised

  • attribute (str, optional) – The attribute field to rasterise, by default “FID”

  • otype (str, optional) – GDAL output type, by default “MEM” (in memory). Set to GTIff for file output

  • fname_out (str, optional) – The output filename, by default “” (e.g. in memory raster).

Returns

An array with the rasterised raster as a numpy array.

Return type

ndarray
